# 🎹 Piano Calculadora

> ### 🚀 **ACCEDER A LA APP AQUÍ: https://germansartori.github.io/piano-calculadora/**

Esta herramienta permite visualizar intervalos, construir escalas y escuchar acordes de forma inmediata, funcionando como una calculadora visual sobre el teclado.

## 🛠️ Guía Rápida de Uso

1. **Marcar Notas:** Clic en las teclas para seleccionarlas. Se pondrán azules. Ideal para contar semitonos, acordes y escalas.
2. **Escuchar (Play/Pausa):** Clic en el botón verde para activar el sonido. Sonarán todas las notas que estén marcadas.
3. **Interactividad:** Con el Play activado, cualquier tecla que se toque se sumará o restará al sonido en tiempo real.
4. **Limpiar:** Usar "Limpiar Todo" para borrar las marcas y silenciar el piano de un solo golpe.
